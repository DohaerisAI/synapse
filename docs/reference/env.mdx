## LLM Provider

| Variable | Default | Description |
|----------|---------|-------------|
| `CODEX_MODEL` | `gpt-5.4` | Codex model name |
| `CODEX_AUTH_FILE` | `~/.codex/auth.json` | Codex CLI auth file path |
| `CODEX_TRANSPORT` | `responses` | `responses` or `cli` |
| `AZURE_OPENAI_ENDPOINT` | — | Azure endpoint URL |
| `AZURE_OPENAI_API_KEY` | — | Azure API key |
| `AZURE_OPENAI_MODEL` | — | Azure deployment name |
| `AZURE_OPENAI_API_VERSION` | `2024-12-01-preview` | Azure API version |
| `OPENAI_API_KEY` | — | OpenAI-compatible API key |
| `OPENAI_BASE_URL` | — | OpenAI-compatible base URL |
| `OPENAI_MODEL` | — | Model name |

## Telegram

| Variable | Default | Description |
|----------|---------|-------------|
| `TELEGRAM_BOT_TOKEN` | — | Bot token from @BotFather |
| `TELEGRAM_POLLING_ENABLED` | `0` | Enable polling |
| `TELEGRAM_ALLOWED_CHAT_IDS` | — | Comma-separated allowed chat IDs |

## Server

| Variable | Default | Description |
|----------|---------|-------------|
| `SERVER_HOST` | `127.0.0.1` | Bind host |
| `SERVER_PORT` | `8000` | Bind port |

## Memory & Workspace

| Variable | Default | Description |
|----------|---------|-------------|
| `MEMORY_ROOT` | `var/memory` | Memory files directory |
| `WORKSPACE_ROOT` | `.` | Workspace root for SELF.md etc. |

## Heartbeat

| Variable | Default | Description |
|----------|---------|-------------|
| `HEARTBEAT_ENABLED` | `0` | Enable periodic heartbeat |
| `HEARTBEAT_EVERY_MINUTES` | `10` | Heartbeat interval |
| `HEARTBEAT_ACTIVE_HOURS_START` | `9` | Active window start (hour) |
| `HEARTBEAT_ACTIVE_HOURS_END` | `22` | Active window end (hour) |

## Jobs

| Variable | Default | Description |
|----------|---------|-------------|
| `JOB_MAX_CONCURRENCY` | `1` | Max parallel background jobs |

## Execution

| Variable | Default | Description |
|----------|---------|-------------|
| `EXECUTION_ISOLATED_ENABLED` | `0` | Enable Docker sandbox |
| `EXECUTION_DOCKER_IMAGE` | `python:3.12-slim` | Docker image for sandbox |
| `EXECUTION_DOCKER_ALLOW_NETWORK` | `0` | Allow network in Docker |
| `EXECUTION_DOCKER_MOUNT_WORKSPACE` | `0` | Mount workspace in Docker |
| `EXECUTION_TIMEOUT_SECONDS` | `60` | Shell command timeout |
| `EXECUTION_MAX_OUTPUT_BYTES` | `65536` | Max output captured |
| `SKILL_AUTO_INSTALL_DEPS` | `0` | Auto-install skill Python deps |
| `EXECUTION_ENABLE_LIVE_ANALYZE_NL_ROUTER` | `0` | Enable NL live-analysis router |

## File system tools

| Variable | Default | Description |
|----------|---------|-------------|
| `FS_ALLOW_ABSOLUTE` | `0` | Allow paths outside repo root |
| `FS_REQUIRE_APPROVAL` | `0` | Force approval for all fs mutations |

## Google Workspace

| Variable | Default | Description |
|----------|---------|-------------|
| `GWS_ENABLED` | `0` | Enable GWS integration |
| `GWS_ALLOWED_SERVICES` | — | Comma-separated: gmail,calendar,drive,docs,sheets |

## Usage & Pricing

| Variable | Default | Description |
|----------|---------|-------------|
| `PRICING_JSON` | — | Path to pricing config JSON |
